# Code of Conduct

We are students who built this simulator for COE 341 and decided to share it. This is not an official course resource, just something we made and thought others might find useful. So this document is not a formal policy. It is more just a few things we think are worth saying.

---

## Be Respectful

If you open an issue to report a bug or ask a question, please keep it straightforward and respectful. We will help where we can, but we are students too and have limited time.

If you spot something wrong in our implementation, we genuinely want to know. Just say it plainly, no need to be harsh about it.

---

## Academic Integrity

This is the part that actually matters. Forking this repo and building on top of it is completely fine. Submitting it as-is for an assignment is not. Whatever you hand in should reflect your own understanding and the work you added, not just the code we wrote.

We are not responsible for how you use this in an academic context. That part is on you.

---

## Using the Issue Tracker

Use GitHub Issues to report bugs in the base simulator. Please do not use them to ask us to debug your own code or help with your assignment. If you find something broken in what we originally shipped though, please do report it because it helps everyone else who forks this.

---

That is really all there is to it. Be decent, be honest, and do your own work.
